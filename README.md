# Image Converter

A lightweight, browser-based image conversion tool with Apple-inspired design.

## Features

- **Format Conversion** - Convert between PNG, JPEG, and WebP
- **Image Compression** - Adjustable quality (10%-100%)
- **Resize** - Scale by percentage or exact pixels
- **Batch Processing** - Process multiple images at once
- **Privacy First** - All processing happens locally in your browser
- **Dark Mode** - Auto-switching based on system preference

## Quick Start

Open `index.html` in any modern browser, or serve locally:

```bash
npx serve .
```

## Usage

1. Drag & drop images or click to select files
2. Choose output format and quality settings
3. Optionally resize images
4. Click "Convert" to download

## Browser Support

Chrome 80+, Safari 14+, Edge 80+, Firefox 75+
